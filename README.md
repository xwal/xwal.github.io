## Installation

### Install Hexo

```shell
$ fnm install lts-iron
$ rbenv install
$ corepack enable
$ yarn global add hexo-cli
```

### Install Dependency

```shell
$ yarn install
$ bundle install
```

### Upgrade Dependency to Latest Version

```shell
$ yarn upgrade --latest
```
